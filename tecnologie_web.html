Tecnologie WEB

HTML

Per la creazione di pagine. Linguaggio di markup, interpretato dal browser.

CSS

Fogli di stile statici. Interpretati dal browser (?).

Less

Fogli di stile dinamici. Si può parametrizzare con una variabile tutti i colori di fondo, in centinaia di fogli di stile e poi a runtime valorizzare quella variabile diversamente (es: JavaScript) e modificare tutto in un colpo solo.

JavaScript

Automatismo lato client. Viene eseguito sul client.

JQuery

Per velocizzare Javascript. Fa la stessa cosa di JS ma in modo diverso, più veloce, più pratico.

JQuery Mobile

JQuery con pacchetti preconfezionati per Mobile.

AngularJS

Per semplificare l'interazione col server.

Angular 2.0

Evoluzione - totalmente diversa - da AngularJS.

Ajax

Permette un'interazione col server in maniera asincrona (concetto simile all'AsyncTask di Android).

Note

JS, JQuery e Angular (2.0) sono componenti (in alternativa, per caratteristiche) di un'app a pagina singola lato client.
Nel momento in cui occorre interagire con il server, se serve ricaricare la pagina, si possono continuare ad utilizzare loro. Ogni chiamata cambia URL e quindi la nuova pagina viene caricata.
Nel caso invece in cui occorra notificare il server ma non ricaricare la pagina (es: carrello Amazon mentre si continua a navigare), si usa AJAX. L'URL rimane sempre la stessa e l'interazione con il server è in backgroud.
