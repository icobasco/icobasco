npm Per gestione moduli nodejs Libreria, contenitore di moduli. Es: voglio comprimere immagini o robe simili, uso un modulo che già lo fa. Uso npm per scaricare. IDE: eclipse, atom, webstorm Eclipse for npm Help/Eclispe Marketplace Nodejs è molto veloce. Se ho db con 7milioni di righe e voglio tirare su i dati con PHP sclera, mentre Nodejs è velocissimo [perchè?]. In php viene mandata in esecuzione la prima funzione, si aspetta che termini e poi viene eseguita la seconda funzione. Nodejs no, riesce a fare le cose in contemporanea.
